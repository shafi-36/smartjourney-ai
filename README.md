# SmartJourney AI ✈️

An AI-powered travel planning assistant that creates personalized itineraries, discovers real-time transportation options, compares fares and discounts, and helps travelers make budget-friendly decisions.

## 🚀 Features

* Personalized travel itinerary generation
* AI-powered travel recommendations
* Real-time transportation discovery
* Fare comparison and discount analysis
* Budget-aware trip planning
* Day-wise travel schedules
* Interactive user interface
* Fast and intelligent trip optimization

## 📌 Problem Statement

Travelers often spend significant time searching across multiple platforms to compare transportation options, prices, and discounts while manually organizing their itineraries. SmartJourney AI simplifies this process by combining AI-driven planning with real-time travel information into a single intelligent platform.

## ⚙️ How It Works

1. User enters destination, budget, duration, and preferences.
2. SmartJourney AI analyzes the requirements.
3. Transportation options are fetched from available APIs.
4. Ticket prices and discounts are compared.
5. Gemini generates a personalized itinerary.
6. The user receives a complete travel plan with estimated costs and recommendations.
